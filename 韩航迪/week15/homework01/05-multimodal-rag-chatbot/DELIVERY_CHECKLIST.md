## ⚠️ 注意事项

### 已知限制

1. **首次启动较慢** - 需要下载AI模型（约1-2GB）
2. **Kafka依赖** - 如不使用异步处理可忽略
3. **网络要求** - 需要访问外网下载模型和调用Qwen API

### 配置修改

如需修改配置，编辑 `main.py`：
- Line 38-40: Milvus连接信息
- Line 79-81: Qwen API密钥
- Line 88: Kafka地址

### 性能建议

1. 使用GPU加速模型推理
2. 增加Milvus索引优化
3. 添加Redis缓存层
4. 使用更小的模型（如bge-tiny）
