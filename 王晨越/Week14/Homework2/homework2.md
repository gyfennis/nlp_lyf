## 二、 交易时机分析（量化动力学模型）

### 1. 买入因子逻辑
| 因子类型 | 判定条件 | 交易建议 |
| :--- | :--- | :--- |
| **能量蓄势** | 周波动率 < 1.5% 且成交量萎缩至 20 日均值以下 | **底部潜伏**，等待波动率拐点 |
| **共振突破** | 日波动率突破周线标准差 + 价格站稳 5 日线 | **确认买入**，波动率进入扩张期 |
| **乖离修复** | 价格跌破周波动下轨后出现首根放量阳线 | **超跌反弹**，轻仓试探 |
| **多头排列** | 日/周波动率同步缓慢上升 + 均线系统多头 | **趋势加仓**，享受波动溢价 |

### 2. 卖出因子逻辑
| 因子类型 | 判定条件 | 交易建议 |
| :--- | :--- | :--- |
| **波动过热** | 日波动率 > 过去 60 日均值 3 倍 | **分批离场**，防范极端情绪反杀 |
| **缩量滞涨** | 价格创新高但日波动率连续 3 日萎缩 | **顶背离预警**，降低仓位 |
| **波动杀跌** | 阴线跌破周波动中轴 + 波动率异常放大 | **止损出局**，波动方向转入下行 |
| **均线死叉** | 5 日线穿透 20 日线且日波动率处于高位 | **清仓信号**，趋势彻底终结 |


### 三、 调用方法
```python
import matplotlib.pyplot as plt
import pandas as pd

def plot_resonance_analysis(data):
    """
    绘制波动共振分析图
    data: 包含 'Daily_Vol', 'Weekly_Vol', 'Signal' 的 DataFrame
    """
    fig, ax1 = plt.subplots(figsize=(12, 6))
    
    # 绘制周波动背景
    ax1.fill_between(data.index, data['Weekly_Vol'], alpha=0.3, color='gray', label='周波动基准线')
    
    # 绘制日波动波动
    ax1.plot(data.index, data['Daily_Vol'], color='blue', linewidth=1, label='日波动率')
    
    # 标注买点
    buy_signals = data[data['Signal'] == 'BUY']
    plt.scatter(buy_signals.index, buy_signals['Daily_Vol'], marker='^', color='red', s=100)
    
    plt.title("波动共振决策图", fontsize=15)
    plt.legend()
    plt.show()
```
