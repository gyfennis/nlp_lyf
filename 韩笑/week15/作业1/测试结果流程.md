## 一、评价函数

### 1.1 Jaccard相似度
```python
def calculate_jaccard_similarity(text1: str, text2: str) -> float:
    set1 = set(text1.lower())
    set2 = set(text2.lower())
    intersection = len(set1 & set2)
    union = len(set1 | set2)
    return intersection / union if union > 0 else 0.0
```

### 1.2 评价函数
```python
def evaluate_answer(
    predicted_answer: str,
    expected_answer: str,
    expected_page: int,
    predicted_pages: List[int],
    expected_file: str,
    predicted_files: List[str]
) -> dict:
    # 1. 页面匹配度 (0.25分) - expected_page in predicted_pages
    # 2. 文件名匹配度 (0.25分) - expected_file basename in predicted_files basenames
    # 3. 答案内容相似度 (0.5分) - Jaccard * 0.5
```

### 1.3 评价指标

| 指标 | 满分 | 计算方式 |
|------|------|----------|
| 页面匹配度 | 0.25 | 期望页码在预测页码列表中 → 1.0 * 0.25 |
| 文件名匹配度 | 0.25 | 期望文件名在预测文件名列表中 → 1.0 * 0.25 |
| 答案内容相似度 | 0.50 | Jaccard(预测答案, 期望答案) * 0.5 |
| **总分** | **1.00** | 三项之和 |

## 三、测试用例

evaluate_answer(
    predicted_answer="机器学习是人工智能的一个分支...",

## 六、注意事项

1. **Worker需单独启动** - 文档上传后Kafka消息由Worker消费处理
2. **Milvus连接** - 向量检索依赖Zilliz Cloud服务
3. **Mineru解析** - PDF解析依赖远程API（耗时较长）
4. **Qwen API** - 问答生成依赖阿里云百炼API
