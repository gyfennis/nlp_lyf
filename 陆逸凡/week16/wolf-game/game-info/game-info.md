# 狼人杀完整规则书（12人标准局）

## 一、游戏配置

### 1.1 阵营与角色（共12人）

| 阵营 | 身份 | 数量 | 阵营内部称号 |
|------|------|------|--------------|
| 狼人阵营 | 狼人 | 4 | 狼队友 |
| 好人阵营 | 村民 | 4 | 平民 |
| 好人阵营 | 预言家 | 1 | 神职 |
| 好人阵营 | 女巫 | 1 | 神职 |
| 好人阵营 | 猎人 | 1 | 神职 |
| 好人阵营 | 白痴 | 1 | 神职 |

**总计**：4狼人 + 4村民 + 4神职 = 12人。

### 1.2 角色技能表

| 角色 | 所属阵营 | 技能名称 | 发动时间 | 效果 | 限制 |
|------|----------|----------|----------|------|------|
| 狼人 | 狼人 | 刀人 | 每夜 | 所有存活的狼人统一意见，击杀一名玩家 | 可自刀（杀队友）；空刀需提前约定 |
| 狼人 | 狼人 | 自爆 | 白天发言阶段 | 立即结束白天，进入黑夜，跳过本日发言和投票 | 只能在未被投票出局时发动；通常每天限爆一次 |
| 预言家 | 好人 | 查验 | 每夜 | 选择一名玩家，获知其身份是**好人**还是**狼人**（不显示具体身份） | 无 |
| 女巫 | 好人 | 救药 | 每夜 | 当晚被杀的人不会死亡 | 只有第一夜可自救；整局只能用一次 |
| 女巫 | 好人 | 毒药 | 每夜 | 选择一名玩家，使其立即死亡（死于毒药） | 整局只能用一次；通常不能两药同夜使用 |
| 猎人 | 好人 | 开枪 | 出局时 | 立刻带走一名存活的玩家 | 被毒死则不能开枪；若最后两人生存且女巫毒死猎人，猎人不能开枪带走另一人 |
| 白痴 | 好人 | 免死 | 被投票放逐时 | 翻开身份牌，免除死亡，但失去投票权 | 仍可发言；狼人仍可在夜里刀他 |

## 二、游戏流程（分阶段详解）

### 2.0 准备阶段
- 法官发放身份牌，所有玩家确认身份。
- 法官宣布 **“天黑请闭眼”**。

### 2.1 夜晚阶段（按顺序执行）

> **注意**：每一个步骤结束后，法官会说“XX请闭眼”，再叫下一个角色。

| 顺序 | 行动方 | 行动内容 | 法官提示语 | 时间 |
|------|--------|----------|------------|------|
| 1 | 全体 | 闭眼 | “天黑请闭眼” | — |
| 2 | 狼人 | 睁眼，统一意见刀一个人（或空刀） | “狼人请睁眼，请确认队友，今晚杀谁？” | 约30秒 |
| 3 | 狼人 | 闭眼 | “狼人请闭眼” | — |
| 4 | 预言家 | 睁眼，指一人要求查验 | “预言家请睁眼，今晚查谁？” | 约10秒 |
| 5 | 预言家 | 法官用手势告知查验结果（拇指向上=狼，拇指向下=好）后闭眼 | “他的身份是……（比手势），预言家请闭眼” | — |
| 6 | 女巫 | 睁眼，法官告知昨夜死亡信息（指认死者） | “女巫请睁眼，今晚死的是他（指人），你有一瓶解药和一瓶毒药……” | 约20秒 |
| 7 | 女巫 | 决定是否用救药或毒药（手势：点头用解药，摇头不用；竖拇指并指人用毒药）后闭眼 | “女巫请闭眼” | — |

> **女巫特殊规则**：
> - 第一夜可用救药自救（通常默认允许）。
> - 救药和毒药不可在同一夜使用。
> - 若女巫用过救药，则不再得知死亡信息。

### 2.2 白天阶段

#### 第一步：公布死亡信息
- 法官宣布：“天亮了，昨晚是平安夜（无人死亡）/ 昨晚XX死亡（可报死因）”。
- 死因判词示例：
  - “昨晚XX倒在了血泊中” → 被狼人刀死
  - “昨晚XX中毒身亡” → 被女巫毒死
  - “昨晚XX被枪杀” → 猎人开枪带走（注意：枪杀只在白天触发，不在天亮时公布）
- 死亡玩家**立即离场**（若有遗言可在法官允许后发言）。

#### 第二步：警上竞选（第一天的特殊阶段）
- 法官宣布：“现在开始警上竞选，想要竞选警长的玩家请举手。”
- 竞选的玩家按一定顺序发言（通常随机或按座位号）。
- 所有竞选者发言完毕后，未竞选的玩家进行投票（可弃票）。
- 得票最高者成为警长，获得**1.5票**投票权，并决定每天的发言顺序（顺时针或逆时针）。
- 若平票：进入PK环节，平票者再发言一轮，其余人投票。若再次平票，本局无警长。

#### 第三步：发言与投票
- 法官指定发言顺序（通常由警长决定，若无警长则从死者左边开始）。
- 每人限时发言（通常60-120秒），**禁止插麦、贴脸、场外**（见第六章）。
- 所有玩家发言完毕后，进入**放逐投票**环节。
- 投票方式：法官倒计时“3-2-1，请投票”，所有人同时举手指向一名玩家（或使用号码牌）。
- 得票最多者被**放逐**，立即亮出身份牌并离场，发表**遗言**（通常只有被放逐的人有遗言，部分规则允许第一天夜晚死亡也有遗言）。
- 若出现平票：平票者进行**PK发言**，其余人重新投票；再次平票则**无人出局**（或按房规处理）。

### 2.3 结束阶段
- 每一轮白天结束后，法官检查是否满足胜利条件。
- 若不满足，则重复**夜晚 → 白天**循环。

## 三、胜利条件（屠边局）

> **屠边规则**：狼人只需杀光**所有村民**或**所有神职**即可获胜（不必杀光所有人）。

### 3.1 好人胜利条件
- 所有狼人出局。

### 3.2 狼人胜利条件（满足其一即可）
1. **屠民**：村民阵营全部出局（包括已死亡及被放逐的村民）。
2. **屠神**：神职阵营全部出局（预言家、女巫、猎人、白痴均出局）。
3. **绝对优势**：狼人数量 ≥ 好人数量，且好人无法翻盘（例如：剩余2狼、1民、0神，狼人已获胜）。

### 3.3 特殊情况
- 若最后一狼被放逐，但当晚女巫毒死了最后一民/神，且狼人刀也生效，则结算顺序为：
  - 优先结算夜晚死亡 → 可能同时达到双方胜利条件 → **平局**（极少见，房规可调）。

## 四、遗言规则（常见标准）

| 死亡方式 | 是否有遗言 | 说明 |
|----------|------------|------|
| 第一夜死亡（被刀/被毒） | ✅ 有 | 通常仅限第一夜 |
| 白天被放逐 | ✅ 有 | 总是有 |
| 猎人开枪带走 | ❌ 无 | 被枪杀的人不发言 |
| 其他夜晚死亡（第二夜及以后） | ❌ 无 | 常见房规改为只有首夜有遗言 |
| 白痴被投而翻牌 | 无 | 因为未真正死亡 |

> 部分房规改为**所有夜晚死亡都有遗言**（更娱乐，但会延长游戏）。

## 五、手势与交流规范（面杀专用）

### 5.1 法官手势（面向预言家）
| 手势 | 含义 |
|------|------|
| 拇指向上 | 狼人 |
| 拇指向下 | 好人 |

### 5.2 女巫手势（面向法官）
| 手势 | 含义 |
|------|------|
| 点头 | 使用解药救人 |
| 摇头 | 不用解药 |
| 竖拇指并指方向 | 使用毒药毒杀某人 |

### 5.3 狼人夜间交流（禁止说话，允许手势）
- 比数字：指刀杀几号玩家。
- 画圈并指自己：自刀。
- 手掌下压：空刀。
- 拇指向上：表示“同意”。
- 拇指向下：表示“反对/换一个”。

## 六、违规判定与“贴脸”清单

### 6.1 严禁行为（一经发现可判罚出局或整局重开）

| 违规类型 | 解释 | 示例 |
|----------|------|------|
| **贴脸发言** | 以场外现实事物发誓来证明身份 | “我要是狼我出门被车撞”“我以我的性别发誓” |
| **场外信息** | 利用闭眼时的动静、角度、声音判断身份 | “我闭眼时听到你那边有动作”“你睁眼时咳嗽了一声” |
| **情绪绑架** | 用哭闹、愤怒、道德绑架迫使别人信你 | “我真的很伤心，你们居然不信我”“你不投他我们绝交” |
| **违规传递信息** | 夜间不该睁眼时睁眼、用手势/文字传信息 | 狼人自爆后仍用手势指刀 |
| **超时发言** | 超过规定发言时间仍继续说话 | 法官喊“过”后仍继续说逻辑 |
| **提前投票/改票** | 未倒计时结束就举手或投票后反悔 | — |

### 6.2 合理但需避免的边缘行为（房规自定）
- **划水**：每轮只说“过”“我是好人，不知道”不提供信息。
- **阴阳发言**：“我就不说我是谁，你们猜吧。”
- **场外现实关系**：“我女朋友肯定不是狼。”

## 七、变体规则速查表（常见房规）

| 变体 | 说明 |
|------|------|
| **屠城局** | 狼人必须杀死所有好人（包括所有神+民）才获胜 → 更累，更慢 |
| **444明牌屠边** | 出局必须亮身份（明牌），狼人更难伪装 |
| **暗牌局** | 只有被放逐/开枪死的人亮身份，夜晚死的不亮 → 狼人更有优势 |
| **连续发言** | 每个白天每人都可以多次发言（需举手），次数不限 |
| **闭眼玩家禁止任何动作** | 闭眼时不能用手指、点头、咳嗽、敲桌子等 |
| **白痴规则变体** | 白痴被放逐后仍可投票（极端少见）/ 白痴被刀时不能翻牌 |

## 八、快速流程卡（法官可用）

```
1. 天黑请闭眼
2. 狼人请睁眼 → 杀人 → 狼人请闭眼
3. 预言家请睁眼 → 验人 → 预言家请闭眼
4. 女巫请睁眼（告知死讯）→ 是否用药 → 女巫请闭眼
5. 天亮了，公布死亡信息
6. 【第一天且未警长时】警上竞选 → 投票选警长
7. 顺序发言（警长定顺序）
8. 放逐投票 → 宣布出局 → 亮身份 → 遗言
9. 检查胜利条件 → 未结束则回第1步
```
